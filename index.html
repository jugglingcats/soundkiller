Sound Killer

Mute and restore sounds on Windows 7 when switching between users

Download this project as a tar.gz file Download this project as a .zip file Download the installer

Welcome to Sound Killer.

Does it annoy you when other users on your computer lock their account with music or other sounds still playing? If the answer is Yes!, then Sound Killer is for you...

Sound Killer is a very lightweight Windows 7 application that runs in the background and automatically mutes all applications when you switch user, and unmutes them again when you switch back.

Note that Sound Killer only works on Windows 7 and above (not tested on Windows 8).

Installing Sound Killer

Sound Killer comes with a complete installer / uninstaller. After install and a reboot it will automatically run as users log in. Download Now
